Movie Project — адаптивный учебный Django-проект, демонстрирующий полный цикл работы с базой данных: создание модели, добавление данных через форму и публикация записей на сайте.
Реализовано:
•	Django-проект movie_project и приложение films
•	Модель Film с полями: название, описание, отзыв
•	Страница добавления фильма: безопасная POST-форма с CSRF-защитой и сохранением в SQLite
•	Страница публикации: вывод всех фильмов из базы данных в виде адаптивных карточек
•	Bootstrap подключён локально из static/ (без CDN), “как по классике”
•	Bootstrap Icons подключены локально, использованы иконки в меню и интерфейсе
•	Современный адаптивный UI: hero-блоки, карточки, аккуратные отступы и типографика
•	Подсветка активного пункта меню автоматически по текущему маршруту
•	Уведомления об успехе/ошибках через Django Messages
